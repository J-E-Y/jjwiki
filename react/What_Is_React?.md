**What is React ?**

# Client Side JavaScript Libray

- Created & Maintained by Facebook

- Used to build dynamic user interfaces

- ""Component"" based

- Most popular framework in the indusrty

# Libray or Framework?

- React us "technically" a library but is often refered to as a framework as it is used for many of the same reasons as a framwork.

# Why Learn & Use React ?

- Build a powerful Frontend Ui

- code organization

- reusable components

- Speed & performance

- Makes things much easier

- Very popular in the indusry

# Component Based UI

- Look at each element as individul components

- organized structure

- Logic, output & style

- Components can have "props " & "state"


# Summary

> 비교적 짧은 러닝커브: JavaScript**와 **html**에 대한 기본적인 지식만 갖추고 있다면, 공식 문서에 소개된 튜토리얼을 통해 개발을 시작할 수 있습니다.

> 재사용성, 유지보수에 용이한 구성요소: React**는 **component** 구조로 이루어져 **재사용성**과 **유지보수**에 있어서 탁월합니다.

> Virtual DOM 을 통한 rendering: React**는 **virtual DOM**을 통해 변화를 가상 DOM에 먼저 적용한 후, 비교를 통해 변화된 부분만 render를 하므로 연산을 줄여줍니다.
